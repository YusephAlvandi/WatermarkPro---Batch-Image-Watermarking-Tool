# WatermarkPro - Batch Image Watermarking Tool

A desktop application for adding custom watermarks to multiple images at once.

## Features
- Batch processing of entire folders
- Custom text watermark with font selection
- Color picker for watermark text
- Opacity control (0-100%)
- Multiple position presets (Top-Left, Center, Bottom-Right, etc.)
- Exact X/Y coordinates for precise placement
- Text rotation (0-360 degrees)
- Settings memory (saves your last configuration)


## How to Use
1. Select input folder (your images)
2. Select output folder (where to save)
3. Enter watermark text
4. Customize appearance (font, color, size, opacity)
5. Choose position
6. Click "Start Processing"

## Technology
Built with Python, CustomTkinter, and Pillow.
